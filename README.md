# CyberPDF

Convert text & Markdown into cyberpunk PDFs. No signup. No API. Just clean conversion.

---

## Live Demo

https://tgmanu-cyberpdf-vert.vercel.app/

---

## Features

- Convert plain text → PDF instantly
- **Markdown support** (headings, bold, italic, lists, blockquotes, hr)
- Drag & drop **.txt / .md** files
- Upload file button (same formats)
- **Live preview** modal before download
- Custom **filename**
- Real-time **character / word / line** stats
- **3 page sizes** (A4, Letter, Legal)
- **Portrait / Landscape** orientation
- **Multiple font families** (Sans / Serif / Mono / Orbitron / Poppins / Roboto)
- **Font size** (Small / Medium / Large)
- **3 PDF themes** (Light / Dark / Sepia)
- Optional **header text** on each page
- Toggle **Page numbers**
- Toggle **Line numbers**
- No authentication
- No ads
- No watermarks

---

## Tech Stack

- React (Vite)
- jsPDF (PDF generation)
- CSS (Cyber themed UI)
- Vercel (static deploy)

## Installation

```
git clone https://github.com/tgmanu/cyberpdf.git
cd cyberpdf
npm install
npm run dev
```
